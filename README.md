# Countries-data-all
Library includes all simple data about all country - name, image flag, phone code, iso2, iso3

## Installation
    npm i countries-data-all

## Usage
import to your file
    import { countriesData } from 'countries-data-all'

you will get an array of the form: 

    [
      {
        country: "Abkhazia",
        countryCodes: ["7840"],
        flag: "./src/ab.svg",
        isoCode2: "AB",
        isoCode3: "ABH",
      },
      {
        country: "Afghanistan",
        countryCodes: ["93"],
        flag: "./src/af.svg",
        isoCode2: "AF",
        isoCode3: "AFG",
      },
    ]
